Component Guide

Button Input Search Checkbox Select Dropdown Toggle Toast
다크모드

Button

버튼 컴포넌트는 다양한 타입과 상태를 지원합니다.

Types

<button className="btn btn-primary">Primary</button>
<button className="btn btn-secondary">Secondary</button>
<button className="btn btn-normal">Normal</button>
<button className="btn btn-ghost">Ghost</button>

Input

텍스트 입력 컴포넌트입니다.

States

오류 메시지입니다.

<input type="text" className="input" placeholder="텍스트를 입력해주세요." />
<input type="text" className="input input-error" placeholder="텍스트를 입력해주세요." />

Search

검색 입력 컴포넌트입니다.

Default

검색어를 입력해주세요.

<div className="search-box">
  <svg className="search-icon">...</svg>
  <p className="search-placeholder">검색어를 입력해주세요.</p>
</div>

Checkbox

체크박스 컴포넌트입니다.

States

<label className="checkbox-label">
  <input type="checkbox" className="checkbox-input" />
  <span className="checkbox-text">체크박스</span>
</label>

Select

드롭다운 선택 컴포넌트입니다.

Default

선택하세요 
<select className="select-box">
  <option>선택하세요</option>
</select>

Dropdown

드롭다운 메뉴 컴포넌트입니다.

Default

Menu Item 1 Menu Item 2 Menu Item 3 
<div className="dropdown-container">
  <button className="dropdown-trigger">Dropdown</button>
  <div className="dropdown-menu">
    <a href="#" className="dropdown-item">Menu Item 1</a>
  </div>
</div>

Toggle

토글 스위치 컴포넌트입니다.

States

<label className="toggle-label">
  <input type="checkbox" className="toggle-input" />
  <span className="toggle-slider"></span>
</label>

Toast

토스트 알림 컴포넌트입니다.

Types

✓ 성공 메시지입니다.
✕ 오류 메시지입니다.
ⓘ 정보 메시지입니다. 
<div className="toast toast-success">
  <span className="toast-icon">✓</span>
  <span className="toast-message">성공 메시지입니다.</span>
</div>
